# **App Name**: Echoes at Dusk

## Core Features:

- Venue Check-In: Users check into the designated seaside bar with soft onboarding (nickname, avatar).
- Vibe Tag Selection: Users select up to 3 vibe tags to express their current mood and intent.
- Presence-Based Discovery: Display only currently checked-in users at the same venue in a discovery feed.
- Expiring Profiles: Profiles automatically expire upon checkout or after a set time limit to maintain spontaneity.
- Soft Interaction (Reactions): Users can react to vibe tags or polls to initiate interaction.
- Mutual-Interaction Chat: Chat functionality unlocks only after mutual positive interactions (reactions).
- AI vibe curator: The application will contain an AI powered tool that suggests alternate 'vibe tags' or other bars in the city that match your criteria when no suitable matches are found.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to evoke a sense of mystery and premium nightlife.
- Background color: Dark charcoal grey (#333333) to enhance the dark, coastal aesthetic with low saturation.
- Accent color: Soft lavender (#D8B4F0) to highlight interactive elements.
- Body and headline font: 'Poppins' for a contemporary, precise feel.
- Minimalistic, custom icons representing vibe tags, using soft gradients and blurs.
- Emotion-driven layout with a focus on visual elements (avatars, vibe tags) rather than extensive text.
- Subtle motion effects, such as gentle pulsing glows around avatars and tags, for a premium feel.
